<task type="auto">
  <name>Task 2: Create Drizzle schema with 4 tables</name>
  <files>src/lib/db/schema.ts</files>
  <action>
Create Drizzle schema file with these tables:

**1. periods table** (time periods for monthly data)
- id: text (PK) — format "YYYY-MM" e.g., "2026-03"
- month: integer (1-12)
- year: integer
- pdfFileName: text (original PDF filename)
- extractionVersion: text (version of extraction logic)
- createdAt: timestamp
- Index: year, month

**2. emiten table** (IDX stocks)
- id: text (PK) — stock code (e.g., "TLKM", "BBCA")
- name: text — company name
- sector: text (nullable) — sector if available
- Index: name (for search)

**3. holders table** (shareholders)
- id: serial (PK)
- canonicalName: text (unique) — normalized name for deduplication
- originalName: text — name as it appears in PDF
- type: text enum ('individual', 'institution', 'foreign', 'government', 'other')
- Index: canonicalName, originalName

**4. ownership_records table** (historical ownership data)
- id: serial (PK)
- periodId: text (FK → periods.id)
- emitenId: text (FK → emiten.id)
- holderId: integer (FK → holders.id)
- rank: integer — position in top holders list
- sharesOwned: bigint — number of shares
- ownershipPercentage: decimal(5,2) — percentage with 2 decimals
- createdAt: timestamp
- Composite unique constraint: (periodId, emitenId, holderId)
- Indexes: periodId, emitenId, holderId, (periodId, emitenId)

Use pgSchema from drizzle-orm/pg-core.
  </action>
  <verify>
    <automated>grep -E "export const (periods|emiten|holders|ownershipRecords)" src/lib/db/schema.ts | wc -l</automated>
  </verify>
  <done>Schema file with 4 tables, proper foreign keys, composite unique constraint, and indexes</done>
</task>

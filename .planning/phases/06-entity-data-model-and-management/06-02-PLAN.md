Create `src/lib/repositories/entity-repository.ts` with these 8 functions:

```typescript
import { db, schema } from '../db';
import { eq, and, sql } from 'drizzle-orm';

// Inferred types
export type Entity = typeof schema.entities.$inferSelect;
export type EntityHolder = typeof schema.entityHolders.$inferSelect;

// 1. createEntity — insert new entity, returns created row
export async function createEntity(name: string, description?: string): Promise<Entity>

// 2. findAllEntities — all entities with alias count (LEFT JOIN + GROUP BY)
export async function findAllEntities(): Promise<Array<{
  id: number; name: string; description: string | null;
  aliasCount: number; createdAt: Date;
}>>

// 3. findEntityById — single entity by ID
export async function findEntityById(id: number): Promise<Entity | null>

// 4. findEntityByHolderId — which entity owns this holder (for conflict message)
export async function findEntityByHolderId(holderId: number): Promise<{ id: number; name: string } | null>

// 5. findAliasesByEntityId — all holder aliases for an entity
export async function findAliasesByEntityId(entityId: number): Promise<Array<{
  holderId: number; holderName: string; holderType: string; addedAt: Date;
}>>

// 6. addAlias — insert into entity_holders (throws PG error 23505 on duplicate holder_id)
export async function addAlias(entityId: number, holderId: number): Promise<EntityHolder>

// 7. removeAlias — DELETE from entity_holders WHERE entity_id AND holder_id
//    NEVER touches holders table or ownership_records table
export async function removeAlias(entityId: number, holderId: number): Promise<void>

// 8. searchHoldersByTrigram — pg_trgm similarity search on holders.canonical_name
//    SET pg_trgm.similarity_threshold = 0.2 before query
//    WHERE canonical_name % query (% is the similarity operator)
//    ORDER BY similarity DESC, LIMIT 20
//    Returns: { id, name, type }[]
export async function searchHoldersByTrigram(query: string, limit?: number): Promise<Array<{
  id: number; name: string; type: string;
}>>
```

Implementation notes:
- `findAllEntities`: use `.leftJoin(schema.entityHolders, eq(...)).groupBy(...).orderBy(schema.entities.name)`
- `searchHoldersByTrigram`: use `sql` template literal for `SET pg_trgm.similarity_threshold = 0.2` and `%` operator; minimum query length check (return [] if query.length < 2) happens in the API route, not here
- Do NOT use `db.transaction()` — Neon HTTP driver does not support it
- Import pattern: `import { db, schema } from '../db'` (consistent with ownership-repository.ts)

Create four API route files. Set `export const revalidate = 0` on mutation routes so server components revalidate after mutations.

**1. `src/app/api/entities/route.ts` — POST /api/entities**
- Accepts: `{ name: string (required, 1-100 chars), description?: string (optional, max 500 chars) }`
- Validates with Zod
- Calls `createEntity(name, description)`
- On success: 201 + entity object
- On name conflict (PG error 23505): 409 + `{ error: 'CONFLICT', message: 'Entity dengan nama ini sudah ada' }`

**2. `src/app/api/entities/[id]/aliases/route.ts` — POST /api/entities/[id]/aliases**
- Accepts: `{ holderId: number (positive int) }`
- Validates entity ID from params (parseInt, isNaN check → 400)
- Validates body with Zod
- Calls `addAlias(entityId, holderId)`
- On success: 200 + `{ success: true }`
- On holder conflict (PG error 23505): call `findEntityByHolderId(holderId)` then return 409:
  ```json
  { "error": "CONFLICT", "message": "Holder ini sudah terdaftar di entity \"<name>\"", "ownerEntityId": 5, "ownerEntityName": "<name>" }
  ```
- Error code to catch: `err?.code === '23505'`

**3. `src/app/api/entities/[id]/aliases/[holderId]/route.ts` — DELETE /api/entities/[id]/aliases/[holderId]**
- Validates both entityId and holderId from params
- Calls `removeAlias(entityId, holderId)` — only deletes from entity_holders
- Returns 200 + `{ success: true }`
- If params invalid: 400

**4. `src/app/api/holders/search/route.ts` — GET /api/holders/search?q=**
- Reads `q` from searchParams
- If `q.length < 2`: return 200 + `[]`
- Calls `searchHoldersByTrigram(q, 20)`
- Returns 200 + array of `{ id, name, type }`

Follow the exact pattern from `src/app/api/holders/[id]/export/route.ts` for params destructuring: `{ params }: { params: Promise<{ id: string }> }` then `await params`.

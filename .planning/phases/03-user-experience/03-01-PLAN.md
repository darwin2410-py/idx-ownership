<task type="auto">
  <name>Task 1: Enhance StocksTable with mobile-responsive layout</name>
  <files>src/components/stocks-table.tsx</files>
  <action>
    Update StocksTable component for mobile responsiveness:

    1. **Add sticky first column** for "Kode Saham" column:
       - Add `left-0 sticky z-10 bg-white` classes to first column header
       - Add `left-0 sticky z-10 bg-white` classes to first column cell
       - Add shadow: `after:absolute after:inset-0 after:z-[-1] after:shadow-[0_0_10px_rgba(0,0,0,0.1)]` for depth

    2. **Responsive padding** for mobile:
       - Change px-6 to `px-2 sm:px-4 md:px-6` for tighter spacing on mobile
       - Change py-3/py-4 to `py-2 sm:py-3 md:py-4` for compact rows

    3. **Text size adjustments**:
       - Header text: `text-[10px] sm:text-xs` for mobile readability
       - Cell text: `text-xs sm:text-sm` for smaller screens

    4. **Container enhancements**:
       - Add `max-w-full` to table container to prevent overflow
       - Ensure `overflow-x-auto` is present on wrapper div
       - Add `-mx-4 sm:mx-0` to offset padding on mobile for full-width scroll

    Example sticky column implementation:
    ```tsx
    columnHelper.accessor('code', {
      header: () => (
        <div className="left-0 sticky z-10 bg-white min-w-[100px]">
          Kode Saham
        </div>
      ),
      cell: (info) => (
        <div className="left-0 sticky z-10 bg-white min-w-[100px]">
          <Link
            href={`/stocks/${info.getValue()}`}
            className="text-blue-600 hover:text-blue-800 font-medium"
          >
            {info.getValue()}
          </Link>
        </div>
      ),
    }),
    ```

    5. **Ensure no body scroll**:
       - Verify table has `overflow-x-auto` not `overflow-auto` (prevent vertical overflow)
       - Container should constrain to viewport width
  </action>
